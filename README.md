# COVID-19 Global Analytics Dashboard (Tableau)

## 📊 Project Overview

This project presents an interactive data visualization dashboard analyzing global COVID-19 trends using Tableau. The dashboard highlights key insights related to cases, deaths, and vaccinations across countries and continents.

The goal of this project is to demonstrate data storytelling and visualization skills by transforming raw data into meaningful insights that support analytical decision-making.

## 📈 Key Insights Visualized

The dashboard explores several important global COVID-19 trends:

• Countries with the highest infection rates
• Death counts across continents
• Global case trends over time
• Vaccination progress compared to population
• Regional comparisons of COVID impact

These visualizations allow users to easily identify patterns and understand how the pandemic evolved across different regions.

## 📂 Data Source

The dataset used in this project includes global COVID-19 statistics such as:

* Total cases
* New cases
* Total deaths
* Population
* Vaccination statistics

The data was explored and prepared using SQL before being visualized in Tableau.

## 🎯 Project Objective

This project demonstrates how data visualization can transform complex datasets into clear, actionable insights. By using interactive dashboards, users can explore trends and better understand the global impact of COVID-19.
